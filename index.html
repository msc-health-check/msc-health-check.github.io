Operação Health Checker

Visualização dos últimos health checks feitos nos serviços registrados.

Como usar

Para registrar um serviço basta fazer uma requisição para a aplicação conforme o exemplo a seguir.
ex.: 

curl -vvv -X POST 'https://msc-health-check.herokuapp.com/health-check' --data-raw '{"URL": "https://myAppExemplo.com/myHealthCheck", "appName": "myAppName"}'

A cada ciclo de 2h a lista de serviços registrados é limpa e deve-se realizar o registro novamente.
