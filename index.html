A cada ciclo de 2h a lista de serviços registrados é limpa e deve-se realizar o registro novamente.
Sobre
Este serviço tem o intuito de ajudar a quem necessita ter um health checker simples. O serviço é gratuito e sem interesse comercial.
Como Usar

Registrar Serviço

Para registrar um serviço basta fazer uma requisição para a aplicação conforme o exemplo a seguir. Desta forma, a cada 5 minutos será efetuada uma checagem na URL informada.
ex.: 
curl -vvv -X POST 'https://msc-health-check.herokuapp.com/health-check' --data-raw '{"URL": "https://myAppExemplo.com/myHealthCheck", "appName": "myAppName"}'
